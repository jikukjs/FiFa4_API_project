# FiFa4_API_project

## 프로젝트 주제 
> FiFa 온라인 4의 게임 데이터 중 상위 랭커들의 선수 활용기록과 그 선수의 능력치 간의 관계를 회귀분석하여 포지션별 필요한 선수를 잘 선택하도록 하자!
